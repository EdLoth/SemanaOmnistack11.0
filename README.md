:clipboard:**Comandos**

* Instalar React-App

`npm install -g create-react-app`

* Inicializar um App

`create-react-app minha-primeira-app`


**-----------------------------**
**Comandos Native**

* Instalar o Expo-cli

`npm install -g expo-cli`

* Inicializar uma aplicação

`expo init nome_projeto`

* Instalar dependências

`npm install`

* Start na aplicação

`npm start`
